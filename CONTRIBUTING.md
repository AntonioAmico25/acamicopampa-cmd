# Contribuindo

Obrigado por querer contribuir! Siga estas etapas básicas:

1. Crie um fork do repositório.
2. Crie uma branch para sua alteração (`git checkout -b feature/nome-da-funcionalidade`).
3. Faça commit das alterações (`git commit -m 'Adicionar nova funcionalidade'`).
4. Faça push para a branch (`git push origin feature/nome-da-funcionalidade`).
5. Abra um Pull Request.

Certifique-se de que todas as contribuições sigam as boas práticas de codificação e que os testes passem antes de enviar.
